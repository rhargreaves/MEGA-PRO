Mappers with full system integration. Can work with IO. Refer to megaio-app for details about using IO

base	- system stuff
map_fmv - mapper used for megacolor player. Good start for using cartridge IO capabilities
map_smd - regular megadrive mapper

For using custom mapper put mapper.rbf along with ROM in same folder
Also for test purposes mapper can be loaded via USB. Refer to megalink tool for details
