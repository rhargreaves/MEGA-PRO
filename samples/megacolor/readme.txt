MegaColor video player by uniq-games. Adapted for mega-ed-pro by krikzz (stand alone version)

mapper - custom FPGA mapper used for player. it provides dual port data buffer for video streaming and dac interface for audio streaming (moved to mappers-std/map_fmv)
player - Genesis application

Converter can be obtained here: http://krikzz.com/pub/support/mega-everdrive/pro-series/megacolor/
